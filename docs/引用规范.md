# MetaDoc Markdown 引用规范

## 概述

MetaDoc 支持在 Markdown 文档中插入参考文献引用，这些引用会在文档末尾自动生成参考文献列表，并且支持点击跳转到对应的参考文献。

## 引用格式

### 基本语法

在文档正文中使用以下格式插入引用：

```markdown
[[1](#ref-1)]
```

或者多个引用：

```markdown
[[1](#ref-1)][[2](#ref-2)]
```

### 格式说明

- `[1]` - 引用编号（显示在文档中的数字）
- `#ref-1` - 锚点 ID（指向参考文献列表中的对应项）

### 示例

```markdown
深度学习是人工智能的一个重要分支[[1](#ref-1)]。近年来，深度学习在图像识别[[2](#ref-2)]和自然语言处理[[3](#ref-3)]等领域取得了突破性进展。

## References

<div id="ref-1"/>

[1] LeCun, Y., Bengio, Y., & Hinton, G. (2015). Deep learning. Nature, 521(7553), 436-444.

<div id="ref-2"/>

[2] Krizhevsky, A., Sutskever, I., & Hinton, G. E. (2012). Imagenet classification with deep convolutional neural networks. Advances in neural information processing systems, 25.

<div id="ref-3"/>

[3] Devlin, J., Chang, M. W., Lee, K., & Toutanova, K. (2018). Bert: Pre-training of deep bidirectional transformers for language understanding. arXiv preprint arXiv:1810.04805.
```

## 参考文献列表格式

### 基本结构

参考文献列表应该放在文档末尾，使用以下格式：

```markdown
## References

<div id="ref-1"/>

[1] 作者. 标题. 出版社, 年份.

<div id="ref-2"/>

[2] 作者. 标题. 出版社, 年份.
```

### 格式要求

1. **标题**：使用 `## References` 作为参考文献列表的标题
2. **锚点**：每个参考文献前必须有一个 `<div id="ref-X"/>` 锚点，其中 `X` 是引用编号
3. **编号**：使用 `[X]` 格式的编号，其中 `X` 是引用编号
4. **内容**：参考文献的完整信息（作者、标题、出版社、年份等）

### 编号规则

- 引用编号可以是数字（如 `1`, `2`, `3`）或字符串（如 `key1`, `key2`）
- 编号会按照数字优先、然后按字符串排序的方式自动排序
- 建议使用连续的数字编号，便于阅读和管理

## 从 LaTeX 转换

如果从 LaTeX 文档转换，MetaDoc 会自动处理以下格式：

### LaTeX 引用命令

```latex
\cite{1}
\cite{1,2,3}
```

会被自动转换为：

```markdown
[[1](#ref-1)]
[[1](#ref-1)][[2](#ref-2)][[3](#ref-3)]
```

### LaTeX 参考文献

```latex
\begin{thebibliography}{99}
\bibitem{1} 作者. 标题. 出版社, 年份.
\bibitem{2} 作者. 标题. 出版社, 年份.
\end{thebibliography}
```

会被自动转换为：

```markdown
## References

<div id="ref-1"/>

[1] 作者. 标题. 出版社, 年份.

<div id="ref-2"/>

[2] 作者. 标题. 出版社, 年份.
```

## 交互功能

### 点击跳转

- 点击文档中的引用链接（如 `[[1](#ref-1)]`）会自动滚动到对应的参考文献
- 支持在当前页面内平滑滚动，不会打开新的浏览器窗口

### 导出支持

- **Markdown 导出**：保持原始引用格式
- **HTML 导出**：引用链接可点击跳转
- **DOCX 导出**：自动转换为 Word 的交叉引用格式（见下方说明）
- **PDF 导出**：引用链接可点击跳转

## DOCX 导出说明

当导出为 DOCX 格式时，MetaDoc 会自动将引用格式转换为 Word 的交叉引用字段：

### 自动转换

1. **引用链接转换**：
   - `[[1](#ref-1)]` 会被转换为 Word 的交叉引用字段（REF 字段）
   - 引用会显示为上标样式（`[1]`），颜色为蓝色
   - 使用 Word 的 `{ REF ref-1 \h }` 字段代码格式
   - 点击引用可以跳转到对应的参考文献

2. **参考文献锚点转换**：
   - `<div id="ref-1"/>` 会被转换为 Word 书签格式
   - 书签名称为 `ref-1`，用于交叉引用字段的链接目标

3. **样式优化**：
   - 引用编号使用上标样式（`vertical-align: super`）
   - 字体大小调整为 0.83em，使其看起来更像学术引用
   - 链接颜色为蓝色，符合 Word 超链接的默认样式

### Word 交叉引用字段

导出后的 DOCX 文件中，引用会被转换为 Word 的交叉引用字段：

- **字段代码**：`{ REF ref-1 \h }`
  - `REF` - 交叉引用字段类型
  - `ref-1` - 书签名称
  - `\h` - 超链接选项，使引用可点击跳转

- **显示效果**：`[1]`（上标样式，蓝色）

### 在 Word 中使用

1. **查看字段代码**：
   - 选中引用文本，按 `Alt+F9` 可以切换显示字段代码
   - 再次按 `Alt+F9` 切换回显示结果

2. **更新引用**：
   - 右键点击引用，选择"更新域"来更新引用编号
   - 或者按 `Ctrl+A` 全选，然后按 `F9` 更新所有字段

3. **编辑引用**：
   - 切换到字段代码视图（`Alt+F9`）
   - 修改书签名称或字段代码
   - 切换回结果视图（`Alt+F9`）

4. **参考文献列表**：
   - 参考文献列表保持原有格式
   - 每个参考文献前有书签（`ref-1`, `ref-2` 等）
   - 可以通过"插入" -> "书签"查看所有书签

### 最佳实践

1. **导出前检查**：确保所有引用都有对应的参考文献
2. **编号一致性**：使用连续的数字编号，便于在 Word 中管理
3. **格式统一**：保持参考文献格式的一致性
4. **更新字段**：在 Word 中修改文档后，记得更新所有字段（`Ctrl+A` + `F9`）

## LaTeX 导出说明

当导出为 LaTeX 格式时，MetaDoc 会自动将引用格式转换为 LaTeX 的引用命令：

### 自动转换

1. **引用链接转换**：
   - `[[1](#ref-1)]` 会被转换为 `\cite{1}`
   - 多个引用 `[[1](#ref-1)][[2](#ref-2)]` 会被转换为 `\cite{1}\cite{2}`

2. **参考文献列表转换**：
   - `## References` 部分会被转换为 `\begin{thebibliography}{99}...\end{thebibliography}` 环境
   - 每个 `<div id="ref-1"/>` 和 `[1] 内容` 会被转换为 `\bibitem{1} 内容`

### LaTeX 格式示例

**Markdown 输入**：
```markdown
深度学习是人工智能的重要分支[[1](#ref-1)]。

## References

<div id="ref-1"/>

[1] Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep Learning. MIT Press.
```

**LaTeX 输出**：
```latex
深度学习是人工智能的重要分支\cite{1}。

\begin{thebibliography}{99}
\bibitem{1} Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep Learning. MIT Press.
\end{thebibliography}
```

### 在 LaTeX 中使用

1. **编译文档**：
   - 使用 `xelatex` 或 `pdflatex` 编译 LaTeX 文档
   - 引用会自动编号并链接到参考文献列表

2. **自定义样式**：
   - 可以使用 `natbib` 或 `biblatex` 包来自定义引用样式
   - 修改 `thebibliography` 环境的格式

3. **引用命令**：
   - `\cite{1}` - 基本引用
   - `\cite{1,2,3}` - 多个引用（需要在 Markdown 中使用 `[[1](#ref-1)][[2](#ref-2)][[3](#ref-3)]`）

## 最佳实践

1. **统一编号**：建议使用连续的数字编号（1, 2, 3...）
2. **完整信息**：确保参考文献包含完整的作者、标题、出版社、年份等信息
3. **格式一致**：保持参考文献格式的一致性，便于阅读
4. **及时更新**：添加或删除参考文献后，记得更新引用编号

## 注意事项

1. 锚点 ID 必须与引用链接中的 ID 完全匹配（区分大小写）
2. 参考文献列表必须包含所有被引用的编号
3. 建议将参考文献列表放在文档末尾
4. 多个引用可以连续使用，如 `[[1](#ref-1)][[2](#ref-2)]`

## 示例文档

完整的示例文档：

```markdown
# 深度学习研究综述

深度学习是机器学习的一个分支，它试图模拟人脑神经网络的结构和功能[[1](#ref-1)]。

## 卷积神经网络

卷积神经网络（CNN）在图像识别任务中表现出色[[2](#ref-2)]。LeNet 是最早的 CNN 架构之一[[3](#ref-3)]。

## 循环神经网络

循环神经网络（RNN）特别适合处理序列数据[[4](#ref-4)]。LSTM 是 RNN 的一个重要变体[[5](#ref-5)]。

## References

<div id="ref-1"/>

[1] Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep Learning. MIT Press.

<div id="ref-2"/>

[2] Krizhevsky, A., Sutskever, I., & Hinton, G. E. (2012). Imagenet classification with deep convolutional neural networks. Advances in neural information processing systems, 25.

<div id="ref-3"/>

[3] LeCun, Y., Bottou, L., Bengio, Y., & Haffner, P. (1998). Gradient-based learning applied to document recognition. Proceedings of the IEEE, 86(11), 2278-2324.

<div id="ref-4"/>

[4] Rumelhart, D. E., Hinton, G. E., & Williams, R. J. (1986). Learning representations by back-propagating errors. Nature, 323(6088), 533-536.

<div id="ref-5"/>

[5] Hochreiter, S., & Schmidhuber, J. (1997). Long short-term memory. Neural computation, 9(8), 1735-1780.
```
