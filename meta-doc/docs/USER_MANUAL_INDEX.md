### 4.5 Agent工具

- **文档编辑类工具**
  - 文档编辑工具（edit-tool）：直接对文章进行编辑，支持多处编辑、插入文本、替换文本、删除文本，支持基于位置的编辑和基于查找替换的编辑
  - 元信息管理工具（metadata-tool）：访问、修改文档元信息（标题、作者、描述、关键词），支持调用AI工具生成元信息
  - 标题格式化工具（title-format-tool）：格式化文档标题，调整标题层级、添加/移除标题编号、标题格式规则应用

- **搜索检索类工具**
  - RAG检索工具（rag-tool）：在知识库中搜索相关内容，支持向量搜索和混合检索，可设置相似度阈值
  - 文本搜索工具（grep-tool）：在当前文档和元信息中搜索文本/正则表达式，返回匹配内容和上下文，支持模糊搜索、正则表达式搜索

- **文本处理类工具**
  - 文本校对工具（proofread-tool）：检查文本中的语法错误、拼写错误、LaTeX语法错误，提供修改建议，支持错误分类（语法、拼写、LaTeX、风格等）
  - 文本比对工具（diff-tool）：比对两段文本、URL或文件，返回差异结果，支持Unified diff格式，显示插入、删除、替换等变更

- **数据分析类工具**
  - 数据分析工具（data-analysis-tool）：自动分析CSV、JSON、XLS、XLSX等结构化数据，提供字段信息提取、描述统计、聚合分析、数据报告生成
  - 数据计算工具（calculation-tool）：执行复杂的数学计算，包括数值运算、向量运算、矩阵运算、统计计算、表达式求值

- **图表生成类工具**
  - 图表生成工具（chart-generation-tool）：根据描述生成各种类型的图表代码（Mermaid、ECharts、PlantUML、flowchart、graphviz等），支持多种图表类型

- **大纲管理类工具**
  - 大纲树工具（outline-tree-tool）：获取当前文档的大纲树结构，支持选择是否包含文本内容，可用于文档结构分析
  - 大纲优化工具（outline-optimize-tool）：基于文档大纲生成/优化大纲结构，支持生成子章节、生成章节内容、大纲重构、大纲结构调整
  - 任务列表工具（todolist-tool）：根据用户输入生成结构化的任务列表，支持任务划分、意图识别、任务管理

- **系统工具类工具**
  - 终端执行工具（terminal-tool）：执行终端/命令行指令，执行前需要用户批准（可设置信任模式），支持命令执行结果分析
  - 网页访问工具（web-crawler-tool）：访问指定URL，获取网页HTML内容或API响应，支持HTTP请求（GET/POST/PUT/DELETE），可绕过CORS限制
  - 工作区文件读取工具（workspace-tool）：读取工作区文件夹中的文件，支持路径字符串和行数范围（如"path/to/file.txt:L123-L456"），支持文件内容摘要
  - 工具说明获取器工具（tool-spec-fetcher-tool）：动态获取一个或多个工具的完整说明文档，用于AI了解工具的使用方法

- **辅助工具类工具**
  - 颜色处理工具（color-tool）：处理颜色混合、亮度/对比度调整、互补色获取、颜色格式转换（HEX、RGB、HSL）、颜色分析
  - 时间戳工具（timestamp-tool）：返回当前时间戳，记录每次调用的时间，可用于计算时间差、监控任务执行时间
