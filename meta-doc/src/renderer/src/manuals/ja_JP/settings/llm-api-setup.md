# LLM API 設定ガイド ✨

MetaDoc でチャット、校正、エージェントを使うには、設定で**動作する LLM API** を用意する必要があります 📡。このページでは、概念 → MetaDoc での入力 → 代表的なプロバイダーの有効化 → トラブルシューティング → 初心者向け FAQ の順で説明します。

> **⚠️ 免責事項**：ローカルの **Ollama** 🦙 を除き、OpenRouter、OpenAI、Google、Alibaba Cloud、DeepSeek、**4sapi** などの集約サービスは**第三者サービス**です。課金、コンプライアンス、アカウントの安全性、プライバシーは利用者と各プロバイダーとの関係にあります。MetaDoc は設定されたエンドポイントに対して互換リクエストを送るクライアントにすぎません。

## 🌐 トラック A — OpenRouter（ワンストップ、OpenAI 互換）

OpenRouter は多くのモデルを **OpenAI 互換** の Base URL の背後に集約します—覚える URL を一つにしたいときに便利 🎯。

### 1️⃣ 登録

[OpenRouter](https://openrouter.ai/) を開き、アカウントを作成します。

### 2️⃣ API キーを作成

[Keys](https://openrouter.ai/settings/keys) でキーを作成し、**安全な場所にコピー**（多くのプロバイダーは完全なシークレットを一度だけ表示）📋。

### 3️⃣ MetaDoc に入力

**OpenAI 互換** を選び、**Base URL** を `https://openrouter.ai/api/v1` に設定し、キーを貼り、**モデル** に `openrouter/free` やカタログのモデル ID を指定。参考：[OpenRouter ドキュメント](https://openrouter.ai/docs/)、[無料ルーター](https://openrouter.ai/openrouter/free)。

無料ルートでも日次上限やキューがあり得ます ⏳。本番利用ではコンソールの**残高とレート制限**に注意。

---

## 🐋 トラック B — DeepSeek（コスパ）

### 1️⃣ コンソール

[deepseek.com](https://www.deepseek.com/) にサインインし、コンソールの API セクションを開きます。

### 2️⃣ API キー

**API keys**（または同等）ページでキーを作成し、安全に保管 🔐。

### 3️⃣ MetaDoc の項目

**DeepSeek** を選び、キーを貼り、モデルは最新の [API ドキュメント](https://api-docs.deepseek.com/) に従い `deepseek-chat` または `deepseek-reasoner` など。

---

## 🔗 トラック C — 4sapi などの集約（OpenAI 互換）

多くは OpenAI と同じ URL 形（`.../v1/chat/completions`）🌏。[4sapi ドキュメント（Apifox）](https://4sapi.apifox.cn/) で価格とエンドポイントを確認し、キーを作成し、**Base URL** と **モデル ID** をドキュメントどおりコピー。MetaDoc では **OpenAI 互換** を選んで各項目を貼り付け。ルールは頻繁に変わる—**ベンダーコンソールを正**とし、**キーを保護** ⚠️。

---

## 🦙 トラック D — Ollama（ローカル）

### 1️⃣ インストール

[ollama.com](https://ollama.com/) からダウンロードし、アプリが動作することを確認。

### 2️⃣ モデルを pull

例：`ollama pull llama3.1`（現在のモデル名はサイトで確認）。モデルは大きい—十分なディスク容量を 💾。

### 3️⃣ MetaDoc の設定

**Ollama** を選び、ベースを `http://localhost:11434/api`、**モデル** を pull した名前に。離散 GPU があると大幅に速い 🎮。CPU のみは長文で遅くなりがち。詳細は [Ollama on GitHub](https://github.com/ollama/ollama)。

---

## ☁️ トラック E — Alibaba Qwen / DashScope

[DashScope ヘルプ](https://help.aliyun.com/zh/dashscope/) に従い **DashScope** を有効化し API キーを作成。MetaDoc で **Qwen** を選び、コンソールに表示される **OpenAI 互換** Base URL と正しいモデル名、必要ならリージョンを入力。

---

## 🔷 トラック F — Google Gemini

[Google AI for Developers](https://ai.google.dev/)（または Cloud コンソールの手順）で API キーを作成。MetaDoc の **Gemini** で、現在のモデル一覧からキーとモデル ID を入力。

## 🧯 トラブルシューティング

- **401 / 403** 🔐：キー誤り、モデル権限なし、プロジェクト未承認。
- **402 / 課金** 💳：クレジット追加または資金のあるキーに切り替え。
- **429** ⏱️：レート制限—後で再試行、同時実行を減らす、プラン変更。
- **502 / ゲートウェイ** 🌐：上流またはネットワーク—再試行または経路変更。
- **空の出力 / JSON エラー** 🧩：Base URL（`/v1` 含む）、モデル ID の綴り、プロキシ／ファイアウォールを再確認。

## ❓ FAQ

### API キーは何のため？ 🔑

プロバイダーが課金と認可に使う**資格情報**です。MetaDoc は設定したエンドポイントにのみ送ります。公開共有やリポジトリへのコミットは避けてください。漏洩するとクォータや課金を使い切られる可能性があります。

### 課金とトークンは？ 💰

多くのクラウド API は **トークン**（モデルのトークナイザー後のテキスト単位）で課金。入力と出力は別価格のことが多い。**料金、無料枠、バンドル**は各ベンダーの価格ページを参照。MetaDoc はモデルアクセスを再販しません。ローカル Ollama は通常クラウドのトークン課金は避けられますが、電力とハードウェアはかかります。

### Base URL とは？ 🔗

クライアントが呼び出す **HTTP の接頭辞**（ホスト + パス）です。「OpenAI 互換」でベンダーを変えるときは多くの場合 Base URL とモデル ID だけ変更；タイプミスは別サービスや 404/401 の原因になります。

### なぜ「無制限」のクラウドモデルは同梱されない？ ☁️

推論にはコストがあり、地域ポリシーに従う必要があります。MetaDoc は**クライアント**として、ライセンスされたプロバイダーを選び、全ユーザーにコストを隠さず直接支払う形にしています。将来、公式エージェント経由で大規模モデルサービスを提供する可能性はあります。

### 組み込み無料トライアルと自分のキー 🎁

トライアルは通常**小さなクォータ**で、キューやスロットルがあることがあります。自分のキーはそのベンダーとの契約に紐づき—日常利用に向きます。

### ドキュメントのテキストはどこへ？ 🔒

**クラウド API** では、選択したプロバイダー（および**その**プライバシーポリシーに記載の下請け）へ送られます。**ローカル Ollama** を localhost にバインドした場合、内容は通常マシン内に留まりますが、OS とネットワーク設定次第—信頼できないネットワークでは機密を送らないでください。

### HTTP エラーと「設定を確認」の失敗は同じ？ 🧪

必ずしもそうではありません。**設定を確認** は複数のプローブを実行します；素の **401/429** は上記の箇条書きに対応。複数失敗時はキー、Base URL、モデル ID をまとめて再確認。

### Ollama とクラウドの選び方 ⚖️

**Ollama** 🦙 はプライバシーとオフライン寄りの利用で、速度／品質のトレードオフを受け入れる場合。**クラウド** ☁️ は最新の大規模モデルと安定したスループットが欲しく、課金と利用規約を受け入れる場合。

### 1 つのキーを複数 PC で？ 💻

技術的には多くの場合可能ですが、**クォータとレート制限は共有**されます。ベンダーによっては共有や IP を制限—利用規約に従ってください。チームでは別キーまたは IAM 的なポリシーを推奨。

### VPN は必要？ 🌍

使う**プロバイダー**と**ネットワーク**次第。一部の国際エンドポイントは不安定またはブロックされることがあります—自分で接続を試し、現地の規制に従ってください。

---

初回設定はアカウントと長い文字列のコピーが必要ですが、慣れれば「新しいベンダー → 新しい Base URL + モデル」が中心になります ✨。ニーズに合うスタックへの接続、頑張ってください 🎉。
