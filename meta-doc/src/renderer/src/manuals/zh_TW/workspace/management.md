# 工作目錄管理

## 概述

工作目錄管理允許您在 MetaDoc 中開啟和管理資料夾，提供類似檔案管理器的功能。透過工作目錄，您可以方便地瀏覽、開啟和管理專案檔案。

## 工作目錄介紹

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

### 什麼是工作目錄

工作目錄是 MetaDoc 中開啟的一個資料夾，它允許您：

```mermaid
graph LR
    A[開啟資料夾] --> B[工作目錄面板]
    B --> C[瀏覽/開啟檔案]
    C --> D[管理檔案]
    D --> E[重新命名/刪除/新增]
    style A fill:#f3f4f6,stroke:#374151
    style B fill:#f3f4f6,stroke:#374151
    style C fill:#f3f4f6,stroke:#374151
    style D fill:#f3f4f6,stroke:#374151
    style E fill:#f3f4f6,stroke:#374151
```

- **瀏覽檔案**：檢視資料夾中的檔案和子資料夾
- **開啟檔案**：直接在 MetaDoc 中開啟檔案
- **管理檔案**：重新命名、刪除檔案等操作
- **專案組織**：將相關檔案組織在一個目錄中

### 使用場景

工作目錄適用於以下場景：

- **專案管理**：管理專案中的所有文件
- **檔案瀏覽**：快速瀏覽和開啟檔案
- **文件組織**：將相關文件組織在一起
- **批次操作**：對多個檔案進行操作

### 開啟目錄

1.  點擊左側選單的「工作目錄」圖示
2.  如果尚未開啟目錄，會彈出目錄選擇對話方塊
3.  選擇要開啟的資料夾
4.  目錄會在側邊欄中顯示

您可以透過側邊欄存取工作目錄檢視：

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

<ViewMenuItemsDemo mode="demo" :items='["editor", "outline", "home"]' />

### 切換目錄

如果需要切換到其他目錄：

1.  點擊工作目錄標題列的選單按鈕
2.  選擇「開啟資料夾」
3.  選擇新的資料夾
4.  新目錄會取代目前目錄

### 關閉目錄

可以關閉目前開啟的工作目錄：

1.  點擊工作目錄標題列的選單按鈕
2.  選擇「關閉工作目錄」
3.  工作目錄面板會隱藏

## 檔案瀏覽

<ViewMenuItemsDemo mode="demo" :items='["workspace", "editor", "outline"]' />

### 目錄樹結構

工作目錄以樹狀結構顯示：

- **資料夾**：顯示資料夾圖示，可以展開/摺疊
- **檔案**：顯示檔案圖示，顯示檔案名稱
- **層級結構**：支援多級資料夾巢狀

### 展開和摺疊

- **展開資料夾**：點擊資料夾圖示或名稱
- **摺疊資料夾**：再次點擊已展開的資料夾
- **全部展開**：右鍵選單可以選擇全部展開
- **全部摺疊**：右鍵選單可以選擇全部摺疊

### 檔案類型識別

工作目錄會識別檔案類型：

- **Markdown 檔案** (.md)：顯示 Markdown 圖示
- **LaTeX 檔案** (.tex)：顯示 LaTeX 圖示
- **圖片檔案** (.png, .jpg 等)：顯示圖片圖示
- **其他檔案**：顯示通用檔案圖示

## 檔案操作

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

### 開啟檔案

有多種方式可以開啟檔案：

- **雙擊檔案**：雙擊檔案圖示或名稱
- **右鍵選單**：右鍵點擊檔案，選擇「開啟」
- **拖拽**：將檔案拖拽到編輯器區域

開啟檔案後，檔案會在新的標籤頁中開啟。

### 預覽檔案

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

可以預覽檔案而不開啟：

- **右鍵選單**：右鍵點擊檔案，選擇「預覽」
- **預覽模式**：檔案在預覽標籤頁中開啟
- **切換編輯**：預覽模式下可以切換到編輯模式

### 重新命名檔案

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

1.  右鍵點擊要重新命名的檔案
2.  選擇「重新命名」
3.  輸入新檔案名稱
4.  按 Enter 確認，或按 Esc 取消

**注意事項**：

- 重新命名會更改檔案系統中的檔案名稱
- 如果檔案正在編輯，需要先儲存
- 重新命名後檔案路徑會改變

### 刪除檔案

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

1.  右鍵點擊要刪除的檔案
2.  選擇「刪除」
3.  確認刪除操作

**注意事項**：

- 刪除操作不可恢復
- 如果檔案正在編輯，需要先關閉
- 刪除資料夾會刪除其中的所有檔案

### 新增檔案

1.  右鍵點擊資料夾或空白區域
2.  選擇「新增檔案」
3.  輸入檔案名稱（包含副檔名）
4.  按 Enter 確認

新增的檔案會立即在編輯器中開啟。

### 新增資料夾

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

1.  右鍵點擊資料夾或空白區域
2.  選擇「新增資料夾」
3.  輸入資料夾名稱
4.  按 Enter 確認

## 檔案操作進階功能

<ViewMenuItemsDemo mode="demo" :items='["workspace", "editor"]' />

### 複製檔案

1.  右鍵點擊要複製的檔案
2.  選擇「複製」
3.  右鍵點擊目標位置
4.  選擇「貼上」

### 剪下檔案

1.  右鍵點擊要剪下的檔案
2.  選擇「剪下」
3.  右鍵點擊目標位置
4.  選擇「貼上」

### 貼上檔案

1.  複製或剪下檔案後
2.  右鍵點擊目標位置
3.  選擇「貼上」

**注意事項**：

- 貼上到資料夾中會在該資料夾內建立檔案
- 如果目標位置已有同名檔案，會提示覆蓋或重新命名

### 批次操作

可以同時選擇多個檔案進行操作：

- **多選**：按住 Ctrl 鍵點擊多個檔案
- **全選**：使用 Ctrl+A 選擇所有檔案
- **批次操作**：對選中的檔案執行複製、刪除等操作

## 檔案搜尋

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

### 搜尋功能

工作目錄支援檔案搜尋：

1.  在工作目錄面板中，使用搜尋框
2.  輸入檔案名稱或關鍵詞
3.  搜尋結果會高亮顯示

### 搜尋範圍

搜尋會在以下範圍內進行：

- **目前目錄**：目前開啟的工作目錄
- **子目錄**：包括所有子資料夾
- **檔案名稱**：搜尋檔案名稱，不搜尋檔案內容

## 目錄監聽

<ViewMenuItemsDemo mode="demo" :items='["workspace", "outline"]' />

### 自動重新整理

工作目錄會自動監聽檔案系統變化：

- **檔案建立**：新增檔案會自動顯示
- **檔案刪除**：刪除檔案會自動移除
- **檔案重新命名**：重新命名檔案會自動更新
- **檔案修改**：檔案修改會顯示更新標記

### 手動重新整理

如果需要手動重新整理目錄：

1.  右鍵點擊資料夾或空白區域
2.  選擇「重新整理」
3.  目錄會重新載入

## 檔案路徑

### 顯示路徑

工作目錄會顯示檔案的完整路徑：

- **懸停提示**：滑鼠懸停在檔案上顯示完整路徑
- **路徑欄**：某些檢視可能顯示路徑欄
- **右鍵選單**：右鍵選單可能顯示路徑資訊

### 路徑操作

- **複製路徑**：可以複製檔案的完整路徑
- **開啟位置**：可以在檔案管理員中開啟檔案位置
- **路徑導航**：可以透過路徑快速定位檔案

## 最佳實踐

1.  **專案組織**：將相關檔案組織在一個工作目錄中
2.  **檔案命名**：使用清晰的命名規範
3.  **定期備份**：重要檔案定期備份
4.  **檔案清理**：定期清理不需要的檔案
5.  **目錄結構**：保持清晰的目錄結構

## 注意事項

1.  **檔案權限**：確保有檔案的讀寫權限
2.  **檔案鎖定**：某些檔案可能被其他程式鎖定
3.  **路徑長度**：注意檔案路徑長度限制
4.  **特殊字元**：檔案名稱避免使用特殊字元
5.  **檔案大小**：大檔案開啟可能需要時間
