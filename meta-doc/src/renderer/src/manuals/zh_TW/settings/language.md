# 多語言支援

## 概述

MetaDoc支援多語言介面，您可以根據使用習慣選擇合適的語言。切換語言後，介面會立即更新為所選語言。

## 支援的語言

MetaDoc目前支援以下語言：

- **中文簡體** (zh_CN)：預設語言
- **English** (en_US)：英語
- **日本語** (ja_JP)：日語
- **한국어** (ko_KR)：韓語
- **Français** (fr_FR)：法語
- **Deutsch** (de_DE)：德語

## 語言切換

### 切換語言

1. 點擊左側選單底部的語言選擇器
2. 選擇要使用的語言
3. 介面會立即更新為所選語言

### 語言儲存

選擇的語言會自動儲存：

- **自動儲存**：選擇語言後立即儲存
- **下次啟動**：下次啟動應用時會使用上次選擇的語言
- **多視窗同步**：所有視窗會自動同步語言設定

## 介面本地化

### 本地化範圍

語言切換會影響以下介面元素：

- **選單項目**：所有選單和選單項目
- **按鈕文字**：所有按鈕的文字
- **對話方塊**：所有對話方塊和提示訊息
- **設定頁面**：所有設定頁面的標籤和說明
- **錯誤訊息**：錯誤和警告訊息

### 內容語言

語言設定只影響介面語言，不影響：

- **文件內容**：文件內容保持原樣
- **檔案路徑**：檔案路徑保持原樣
- **使用者輸入**：使用者輸入的內容不受影響

## 語言選擇建議

### 根據使用習慣

- **中文使用者**：使用中文簡體，介面更熟悉
- **英文使用者**：使用English，符合使用習慣
- **其他語言**：根據個人偏好選擇

### 根據文件語言

- **中文文件**：可以使用中文介面
- **英文文件**：可以使用英文介面
- **多語言文件**：選擇最常用的語言

## 語言切換效果

### 立即生效

語言切換會立即生效：

- **介面更新**：所有介面元素立即更新
- **無需重啟**：不需要重啟應用
- **狀態保持**：目前編輯狀態不會遺失

<MainTabs mode="demo" />

### 多視窗同步

所有視窗會自動同步語言：

- **主視窗**：主視窗語言切換
- **輔助視窗**：所有輔助視窗同步更新
- **新視窗**：新開啟的視窗使用目前語言

## 語言檔案

### 語言檔案位置

語言檔案儲存在應用目錄中：

- **檔案格式**：JSON格式
- **檔案位置**：`src/renderer/src/locales/`
- **檔案命名**：使用語言代碼命名（如 `zh_cn.json`）

### 語言檔案結構

語言檔案採用鍵值對結構：

```json
{
  "common": {
    "confirm": "確認",
    "cancel": "取消"
  },
  "setting": {
    "basic": "基礎設定"
  }
}
```

## 注意事項

1. **語言代碼**：語言代碼使用底線格式（如 `zh_CN`）
2. **翻譯完整性**：某些新功能可能暫時只有部分語言翻譯
3. **回退語言**：如果翻譯缺失，會回退到中文簡體
4. **文件內容**：語言設定不影響文件內容
5. **檔案路徑**：語言設定不影響檔案路徑顯示
