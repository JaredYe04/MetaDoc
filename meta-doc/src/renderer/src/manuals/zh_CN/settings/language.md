# 多语言支持

## 概述

MetaDoc支持多语言界面，您可以根据使用习惯选择合适的语言。切换语言后，界面会立即更新为所选语言。

## 支持的语言

MetaDoc目前支持以下语言：

- **中文简体** (zh_CN)：默认语言
- **English** (en_US)：英语
- **日本語** (ja_JP)：日语
- **한국어** (ko_KR)：韩语
- **Français** (fr_FR)：法语
- **Deutsch** (de_DE)：德语

## 语言切换

### 切换语言

1. 点击左侧菜单底部的语言选择器
2. 选择要使用的语言
3. 界面会立即更新为所选语言

### 语言保存

选择的语言会自动保存：

- **自动保存**：选择语言后立即保存
- **下次启动**：下次启动应用时会使用上次选择的语言
- **多窗口同步**：所有窗口会自动同步语言设置

## 界面本地化

### 本地化范围

语言切换会影响以下界面元素：

- **菜单项**：所有菜单和菜单项
- **按钮文本**：所有按钮的文本
- **对话框**：所有对话框和提示信息
- **设置页面**：所有设置页面的标签和说明
- **错误信息**：错误和警告信息

### 内容语言

语言设置只影响界面语言，不影响：

- **文档内容**：文档内容保持原样
- **文件路径**：文件路径保持原样
- **用户输入**：用户输入的内容不受影响

## 语言选择建议

### 根据使用习惯

- **中文用户**：使用中文简体，界面更熟悉
- **英文用户**：使用English，符合使用习惯
- **其他语言**：根据个人偏好选择

### 根据文档语言

- **中文文档**：可以使用中文界面
- **英文文档**：可以使用英文界面
- **多语言文档**：选择最常用的语言

## 语言切换效果

### 即时生效

语言切换会立即生效：

- **界面更新**：所有界面元素立即更新
- **无需重启**：不需要重启应用
- **状态保持**：当前编辑状态不会丢失

<MainTabs mode="demo" />

### 多窗口同步

所有窗口会自动同步语言：

- **主窗口**：主窗口语言切换
- **辅助窗口**：所有辅助窗口同步更新
- **新窗口**：新打开的窗口使用当前语言

## 语言文件

### 语言文件位置

语言文件存储在应用目录中：

- **文件格式**：JSON格式
- **文件位置**：`src/renderer/src/locales/`
- **文件命名**：使用语言代码命名（如 `zh_cn.json`）

### 语言文件结构

语言文件采用键值对结构：

```json
{
  "common": {
    "confirm": "确认",
    "cancel": "取消"
  },
  "setting": {
    "basic": "基础设置"
  }
}
```

## 注意事项

1. **语言代码**：语言代码使用下划线格式（如 `zh_CN`）
2. **翻译完整性**：某些新功能可能暂时只有部分语言翻译
3. **回退语言**：如果翻译缺失，会回退到中文简体
4. **文档内容**：语言设置不影响文档内容
5. **文件路径**：语言设置不影响文件路径显示
