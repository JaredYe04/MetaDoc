# 主頁功能

## 概述

主頁是MetaDoc的入口介面，提供快速開始、新建文件、開啟檔案等功能。主頁設計簡潔美觀，幫助您快速開始使用MetaDoc。

## Agent 與推薦提示

主頂部提供與 **Agent 全頁**相同的輸入框（支援 @ 引用檔案/分頁/目錄）。內容僅暫存於本頁，**按傳送**後會開啟 **Agent** 分頁並**建立新工作階段**，再送出該則訊息。

輸入框下方為 **推薦需求**（多則含圖示的範例提示）：

- 可按 **換一批** 隨機刷新
- 每則約 **10～20 秒**會自動隨機替換並有轉場動畫
- 點選會將文字填入輸入框；可用 **Ctrl+Z** / **⌘Z** 還原填入前內容

> 舊版「快速開始」精靈已移除；請用 **新建文件** 或左側選單建立文件。
## 新建文件

### 建立空白文件

點擊「新建文件」按鈕可以快速建立空白文件：

1.  點擊「新建文件」按鈕
2.  選擇文件格式（Markdown/LaTeX/純文字）
3.  文件會在新分頁中開啟

**快速鍵**：也可以使用 `Ctrl+N`（Windows/Linux）或 `Cmd+N`（macOS）快速建立。

## 開啟檔案

### 開啟既有檔案

點擊「開啟檔案」按鈕可以開啟既有檔案：

1.  點擊「開啟檔案」按鈕
2.  在檔案選擇對話方塊中選擇檔案
3.  檔案會在新分頁中開啟

**快速鍵**：也可以使用 `Ctrl+O`（Windows/Linux）或 `Cmd+O`（macOS）快速開啟。

### 支援的檔案格式

-   **Markdown** (.md)
-   **LaTeX** (.tex)
-   **純文字** (.txt)
-   **JSON** (.json)

## 使用者手冊

### 開啟使用者手冊

點擊「使用者手冊」按鈕可以開啟使用者手冊：

1.  點擊「使用者手冊」按鈕
2.  使用者手冊會在新分頁中開啟
3.  可以瀏覽和學習各種功能

**快速鍵**：也可以按 `F1` 鍵快速開啟使用者手冊。

## 最近文件清單

### 檢視最近文件

主頁會顯示最近開啟的文件清單：

-   **顯示數量**：最多顯示12個最近文件
-   **文件卡片**：每個文件顯示為一個卡片
-   **快速開啟**：點擊卡片即可快速開啟文件

### 最近文件操作

-   **開啟文件**：點擊文件卡片開啟文件
-   **刪除記錄**：點擊卡片上的刪除按鈕刪除記錄
-   **右鍵選單**：右鍵點擊卡片可能有更多選項

### 最近文件管理

-   **自動更新**：開啟文件後會自動更新清單
-   **記錄儲存**：最近文件記錄會儲存
-   **清單排序**：按開啟時間倒序排列

## 使用者資料對話方塊

### 開啟使用者資料

主頁可能會顯示使用者資料對話方塊：

-   **首次使用**：首次使用時可能提示設定使用者資料
-   **資料設定**：可以設定使用者畫像和使用偏好
-   **AI優化**：使用者資料可以幫助AI更好地理解您的需求

### 使用者資料內容

使用者資料可能包括：

-   **基本資訊**：姓名、職業等
-   **使用偏好**：編輯習慣、常用功能等
-   **使用者畫像**：幫助AI理解您的使用場景

## 主頁介面

### 介面佈局

主頁採用置中佈局：

-   **頂部**：MetaDoc標題和副標題
-   **中間**：操作按鈕區域
-   **底部**：最近文件清單

### 視覺設計

主頁採用簡潔現代的設計：

-   **動態背景**：動態背景動畫效果
-   **網格裝飾**：極簡網格裝飾
-   **卡片設計**：操作按鈕採用卡片設計

## 最佳實踐

1.  **快速開始**：首次使用建議使用快速開始精靈
2.  **快速鍵**：熟練使用快速鍵提高效率
3.  **最近文件**：利用最近文件清單快速存取常用文件
4.  **使用者資料**：設定使用者資料以獲得更好的AI體驗
5.  **使用者手冊**：遇到問題時檢視使用者手冊

## 注意事項

1.  **主頁顯示**：主頁只在沒有開啟文件時顯示
2.  **快速開始**：快速開始精靈可以隨時關閉
3.  **最近文件**：最近文件清單最多顯示12個
4.  **使用者資料**：使用者資料設定是選用的
5.  **介面語言**：主頁介面語言跟隨系統語言設定
