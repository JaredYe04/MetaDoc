# 工作目录管理

## 概述

工作目录管理允许您在MetaDoc中打开和管理文件夹，提供类似文件管理器的功能。通过工作目录，您可以方便地浏览、打开和管理项目文件。

## 工作目录介绍

### 什么是工作目录

工作目录是MetaDoc中打开的一个文件夹，它允许您：

```mermaid
graph LR
    A[打开文件夹] --> B[工作目录面板]
    B --> C[浏览/打开文件]
    C --> D[管理文件]
    D --> E[重命名/删除/新建]
    style A fill:#f3f4f6,stroke:#374151
    style B fill:#f3f4f6,stroke:#374151
    style C fill:#f3f4f6,stroke:#374151
    style D fill:#f3f4f6,stroke:#374151
    style E fill:#f3f4f6,stroke:#374151
```

- **浏览文件**：查看文件夹中的文件和子文件夹
- **打开文件**：直接在MetaDoc中打开文件
- **管理文件**：重命名、删除文件等操作
- **项目组织**：将相关文件组织在一个目录中

### 使用场景

工作目录适用于以下场景：

- **项目管理**：管理项目中的所有文档
- **文件浏览**：快速浏览和打开文件
- **文档组织**：将相关文档组织在一起
- **批量操作**：对多个文件进行操作

## 打开工作目录

### 打开目录

1. 点击左侧菜单的"工作目录"图标
2. 如果尚未打开目录，会弹出目录选择对话框
3. 选择要打开的文件夹
4. 目录会在侧边栏中显示

### 切换目录

如果需要切换到其他目录：

1. 点击工作目录标题栏的菜单按钮
2. 选择"打开文件夹"
3. 选择新的文件夹
4. 新目录会替换当前目录

### 关闭目录

可以关闭当前打开的工作目录：

1. 点击工作目录标题栏的菜单按钮
2. 选择"关闭工作目录"
3. 工作目录面板会隐藏

## 文件浏览

### 目录树结构

工作目录以树形结构显示：

- **文件夹**：显示文件夹图标，可以展开/折叠
- **文件**：显示文件图标，显示文件名
- **层级结构**：支持多级文件夹嵌套

### 展开和折叠

- **展开文件夹**：点击文件夹图标或名称
- **折叠文件夹**：再次点击已展开的文件夹
- **全部展开**：右键菜单可以选择全部展开
- **全部折叠**：右键菜单可以选择全部折叠

### 文件类型识别

工作目录会识别文件类型：

- **Markdown文件** (.md)：显示Markdown图标
- **LaTeX文件** (.tex)：显示LaTeX图标
- **图片文件** (.png, .jpg等)：显示图片图标
- **其他文件**：显示通用文件图标

## 文件操作

### 打开文件

有多种方式可以打开文件：

- **双击文件**：双击文件图标或名称
- **右键菜单**：右键点击文件，选择"打开"
- **拖拽**：将文件拖拽到编辑器区域

打开文件后，文件会在新的标签页中打开。

### 预览文件

可以预览文件而不打开：

- **右键菜单**：右键点击文件，选择"预览"
- **预览模式**：文件在预览标签页中打开
- **切换编辑**：预览模式下可以切换到编辑模式

### 重命名文件

1. 右键点击要重命名的文件
2. 选择"重命名"
3. 输入新文件名
4. 按Enter确认，或按Esc取消

**注意事项**：
- 重命名会更改文件系统中的文件名
- 如果文件正在编辑，需要先保存
- 重命名后文件路径会改变

### 删除文件

1. 右键点击要删除的文件
2. 选择"删除"
3. 确认删除操作

**注意事项**：
- 删除操作不可恢复
- 如果文件正在编辑，需要先关闭
- 删除文件夹会删除其中的所有文件

### 新建文件

1. 右键点击文件夹或空白区域
2. 选择"新建文件"
3. 输入文件名（包含扩展名）
4. 按Enter确认

新建的文件会立即在编辑器中打开。

### 新建文件夹

1. 右键点击文件夹或空白区域
2. 选择"新建文件夹"
3. 输入文件夹名称
4. 按Enter确认

## 文件操作高级功能

### 复制文件

1. 右键点击要复制的文件
2. 选择"复制"
3. 右键点击目标位置
4. 选择"粘贴"

### 剪切文件

1. 右键点击要剪切的文件
2. 选择"剪切"
3. 右键点击目标位置
4. 选择"粘贴"

### 粘贴文件

1. 复制或剪切文件后
2. 右键点击目标位置
3. 选择"粘贴"

**注意事项**：
- 粘贴到文件夹中会在该文件夹内创建文件
- 如果目标位置已有同名文件，会提示覆盖或重命名

### 批量操作

可以同时选择多个文件进行操作：

- **多选**：按住Ctrl键点击多个文件
- **全选**：使用Ctrl+A选择所有文件
- **批量操作**：对选中的文件执行复制、删除等操作

## 文件搜索

### 搜索功能

工作目录支持文件搜索：

1. 在工作目录面板中，使用搜索框
2. 输入文件名或关键词
3. 搜索结果会高亮显示

### 搜索范围

搜索会在以下范围内进行：

- **当前目录**：当前打开的工作目录
- **子目录**：包括所有子文件夹
- **文件名**：搜索文件名，不搜索文件内容

## 目录监听

### 自动刷新

工作目录会自动监听文件系统变化：

- **文件创建**：新建文件会自动显示
- **文件删除**：删除文件会自动移除
- **文件重命名**：重命名文件会自动更新
- **文件修改**：文件修改会显示更新标记

### 手动刷新

如果需要手动刷新目录：

1. 右键点击文件夹或空白区域
2. 选择"刷新"
3. 目录会重新加载

## 文件路径

### 显示路径

工作目录会显示文件的完整路径：

- **悬停提示**：鼠标悬停在文件上显示完整路径
- **路径栏**：某些视图可能显示路径栏
- **右键菜单**：右键菜单可能显示路径信息

### 路径操作

- **复制路径**：可以复制文件的完整路径
- **打开位置**：可以在文件管理器中打开文件位置
- **路径导航**：可以通过路径快速定位文件

## 最佳实践

1. **项目组织**：将相关文件组织在一个工作目录中
2. **文件命名**：使用清晰的命名规范
3. **定期备份**：重要文件定期备份
4. **文件清理**：定期清理不需要的文件
5. **目录结构**：保持清晰的目录结构

## 注意事项

1. **文件权限**：确保有文件的读写权限
2. **文件锁定**：某些文件可能被其他程序锁定
3. **路径长度**：注意文件路径长度限制
4. **特殊字符**：文件名避免使用特殊字符
5. **文件大小**：大文件打开可能需要时间
