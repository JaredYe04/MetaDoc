# 작업 디렉토리 관리

## 개요

작업 디렉토리 관리를 통해 MetaDoc에서 폴더를 열고 관리할 수 있으며, 파일 관리자와 유사한 기능을 제공합니다. 작업 디렉토리를 통해 프로젝트 파일을 편리하게 탐색, 열기 및 관리할 수 있습니다.

### 작업 디렉토리란

작업 디렉토리는 MetaDoc에서 열린 하나의 폴더로, 다음과 같은 기능을 제공합니다:

```mermaid
graph LR
    A[폴더 열기] --> B[작업 디렉토리 패널]
    B --> C[파일 탐색/열기]
    C --> D[파일 관리]
    D --> E[이름 변경/삭제/새로 만들기]
    style A fill:#f3f4f6,stroke:#374151
    style B fill:#f3f4f6,stroke:#374151
    style C fill:#f3f4f6,stroke:#374151
    style D fill:#f3f4f6,stroke:#374151
    style E fill:#f3f4f6,stroke:#374151
```

- **파일 탐색**: 폴더 내의 파일과 하위 폴더 확인
- **파일 열기**: MetaDoc에서 직접 파일 열기
- **파일 관리**: 이름 변경, 파일 삭제 등의 작업 수행
- **프로젝트 구성**: 관련 파일을 하나의 디렉토리로 구성

### 사용 시나리오

작업 디렉토리는 다음과 같은 시나리오에 적합합니다:

- **프로젝트 관리**: 프로젝트 내 모든 문서 관리
- **파일 탐색**: 빠르게 파일 탐색 및 열기
- **문서 구성**: 관련 문서를 함께 구성
- **일괄 작업**: 여러 파일에 대한 작업 수행

## 작업 디렉토리 열기

### 디렉토리 열기

1. 왼쪽 메뉴의 "작업 디렉토리" 아이콘 클릭
2. 아직 디렉토리가 열려 있지 않다면 디렉토리 선택 대화상자가 표시됨
3. 열고자 하는 폴더 선택
4. 디렉토리가 사이드바에 표시됨

사이드바를 통해 작업 디렉토리 뷰에 접근할 수 있습니다:

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

<ViewMenuItemsDemo mode="demo" :items='["editor", "outline", "home"]' />

### 디렉토리 전환

다른 디렉토리로 전환해야 하는 경우:

1. 작업 디렉토리 제목 표시줄의 메뉴 버튼 클릭
2. "폴더 열기" 선택
3. 새로운 폴더 선택
4. 새 디렉토리가 현재 디렉토리를 대체함

### 디렉토리 닫기

현재 열려 있는 작업 디렉토리를 닫을 수 있습니다:

1. 작업 디렉토리 제목 표시줄의 메뉴 버튼 클릭
2. "작업 디렉토리 닫기" 선택
3. 작업 디렉토리 패널이 숨겨짐

## 파일 탐색

<ViewMenuItemsDemo mode="demo" :items='["workspace", "editor", "outline"]' />

### 디렉토리 트리 구조

작업 디렉토리는 트리 구조로 표시됩니다:

- **폴더**: 폴더 아이콘 표시, 확장/축소 가능
- **파일**: 파일 아이콘 표시, 파일 이름 표시
- **계층 구조**: 다중 레벨 폴더 중첩 지원

### 확장 및 축소

- **폴더 확장**: 폴더 아이콘 또는 이름 클릭
- **폴더 축소**: 이미 확장된 폴더 다시 클릭
- **모두 확장**: 마우스 오른쪽 버튼 메뉴에서 모두 확장 선택 가능
- **모두 축소**: 마우스 오른쪽 버튼 메뉴에서 모두 축소 선택 가능

### 파일 유형 인식

작업 디렉토리는 파일 유형을 인식합니다:

- **Markdown 파일** (.md): Markdown 아이콘 표시
- **LaTeX 파일** (.tex): LaTeX 아이콘 표시
- **이미지 파일** (.png, .jpg 등): 이미지 아이콘 표시
- **기타 파일**: 일반 파일 아이콘 표시

### 파일 열기

파일을 여는 여러 가지 방법:

- **파일 더블클릭**: 파일 아이콘 또는 이름 더블클릭
- **마우스 오른쪽 버튼 메뉴**: 파일 마우스 오른쪽 버튼 클릭, "열기" 선택
- **드래그 앤 드롭**: 파일을 편집기 영역으로 드래그

파일을 열면 파일이 새 탭에서 열립니다.

### 파일 미리보기

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

파일을 열지 않고 미리볼 수 있습니다:

- **마우스 오른쪽 버튼 메뉴**: 파일 마우스 오른쪽 버튼 클릭, "미리보기" 선택
- **미리보기 모드**: 파일이 미리보기 탭에서 열림
- **편집 모드 전환**: 미리보기 모드에서 편집 모드로 전환 가능

### 파일 이름 변경

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

1. 이름을 변경할 파일 마우스 오른쪽 버튼 클릭
2. "이름 변경" 선택
3. 새 파일 이름 입력
4. Enter 키로 확인, 또는 Esc 키로 취소

**주의사항**:

- 이름 변경은 파일 시스템의 파일 이름을 변경합니다
- 파일이 편집 중인 경우 먼저 저장해야 합니다
- 이름 변경 후 파일 경로가 변경됩니다

### 파일 삭제

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

1. 삭제할 파일 마우스 오른쪽 버튼 클릭
2. "삭제" 선택
3. 삭제 작업 확인

**주의사항**:

- 삭제 작업은 되돌릴 수 없습니다
- 파일이 편집 중인 경우 먼저 닫아야 합니다
- 폴더 삭제 시 내부의 모든 파일이 삭제됩니다

### 새 파일 만들기

1. 폴더 또는 빈 영역 마우스 오른쪽 버튼 클릭
2. "새 파일" 선택
3. 파일 이름 입력 (확장자 포함)
4. Enter 키로 확인

새로 만든 파일은 즉시 편집기에서 열립니다.

### 새 폴더 만들기

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

1. 폴더 또는 빈 영역 마우스 오른쪽 버튼 클릭
2. "새 폴더" 선택
3. 폴더 이름 입력
4. Enter 키로 확인

## 파일 작업 고급 기능

<ViewMenuItemsDemo mode="demo" :items='["workspace", "editor"]' />

### 파일 복사

1. 복사할 파일 마우스 오른쪽 버튼 클릭
2. "복사" 선택
3. 대상 위치 마우스 오른쪽 버튼 클릭
4. "붙여넣기" 선택

### 파일 잘라내기

1. 잘라낼 파일 마우스 오른쪽 버튼 클릭
2. "잘라내기" 선택
3. 대상 위치 마우스 오른쪽 버튼 클릭
4. "붙여넣기" 선택

### 파일 붙여넣기

1. 파일 복사 또는 잘라내기 후
2. 대상 위치 마우스 오른쪽 버튼 클릭
3. "붙여넣기" 선택

**주의사항**:

- 폴더에 붙여넣으면 해당 폴더 내에 파일이 생성됩니다
- 대상 위치에 동일한 이름의 파일이 이미 있는 경우 덮어쓰기 또는 이름 변경을 요청합니다

### 일괄 작업

여러 파일을 동시에 선택하여 작업할 수 있습니다:

- **다중 선택**: Ctrl 키를 누른 상태에서 여러 파일 클릭
- **전체 선택**: Ctrl+A를 사용하여 모든 파일 선택
- **일괄 작업**: 선택된 파일에 대해 복사, 삭제 등의 작업 수행

## 파일 검색

<ViewMenuItemsDemo mode="demo" :items='["workspace"]' />

### 검색 기능

작업 디렉토리는 파일 검색을 지원합니다:

1. 작업 디렉토리 패널에서 검색 상자 사용
2. 파일 이름 또는 키워드 입력
3. 검색 결과가 강조 표시됨

### 검색 범위

검색은 다음 범위 내에서 수행됩니다:

- **현재 디렉토리**: 현재 열려 있는 작업 디렉토리
- **하위 디렉토리**: 모든 하위 폴더 포함
- **파일 이름**: 파일 이름 검색, 파일 내용은 검색하지 않음

## 디렉토리 감시

<ViewMenuItemsDemo mode="demo" :items='["workspace", "outline"]' />

### 자동 새로고침

작업 디렉토리는 파일 시스템 변경을 자동으로 감시합니다:

- **파일 생성**: 새 파일이 자동으로 표시됨
- **파일 삭제**: 삭제된 파일이 자동으로 제거됨
- **파일 이름 변경**: 이름이 변경된 파일이 자동으로 업데이트됨
- **파일 수정**: 수정된 파일에 업데이트 표시가 나타남

### 수동 새로고침

수동으로 디렉토리를 새로고침해야 하는 경우:

1. 폴더 또는 빈 영역 마우스 오른쪽 버튼 클릭
2. "새로고침" 선택
3. 디렉토리가 다시 로드됨

## 파일 경로

### 경로 표시

작업 디렉토리는 파일의 전체 경로를 표시합니다:

- **마우스 오버 힌트**: 파일 위에 마우스를 올리면 전체 경로 표시
- **경로 표시줄**: 일부 뷰에서는 경로 표시줄이 표시될 수 있음
- **마우스 오른쪽 버튼 메뉴**: 마우스 오른쪽 버튼 메뉴에 경로 정보가 표시될 수 있음

### 경로 작업

- **경로 복사**: 파일의 전체 경로를 복사할 수 있음
- **위치 열기**: 파일 관리자에서 파일 위치를 열 수 있음
- **경로 탐색**: 경로를 통해 파일을 빠르게 찾을 수 있음

## 모범 사례

1. **프로젝트 구성**: 관련 파일을 하나의 작업 디렉토리로 구성
2. **파일 이름 지정**: 명확한 명명 규칙 사용
3. **정기 백업**: 중요한 파일 정기 백업
4. **파일 정리**: 필요 없는 파일 정기 정리
5. **디렉토리 구조**: 명확한 디렉토리 구조 유지

## 주의사항

1. **파일 권한**: 파일 읽기/쓰기 권한 보유 확인
2. **파일 잠금**: 일부 파일은 다른 프로그램에 의해 잠길 수 있음
3. **경로 길이**: 파일 경로 길이 제한 주의
4. **특수 문자**: 파일 이름에 특수 문자 사용 피하기
5. **파일 크기**: 큰 파일 열기에는 시간이 걸릴 수 있음
