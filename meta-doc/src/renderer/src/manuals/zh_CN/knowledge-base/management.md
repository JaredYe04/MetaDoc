# 知识库管理

## 概述

知识库管理是MetaDoc RAG（检索增强生成）系统的核心功能，允许您将文档添加到知识库中，通过向量搜索为AI对话提供上下文信息。知识库可以帮助AI更好地理解您的文档内容，提供更准确的回答。

## 启用知识库

### 开启知识库功能

在知识库设置页面，首先需要启用知识库功能：

1. 找到"启用知识库"开关
2. 将开关切换到"启用"状态
3. 配置知识库相关参数

启用知识库后，AI功能（如AI对话、AI补全）会自动使用知识库中的信息来增强回答质量。

**注意事项**：
- 启用知识库后，AI功能会检索知识库内容，可能影响响应速度
- 知识库需要先添加文件才能发挥作用
- 建议在添加文件后再启用知识库

## 置信度阈值设置

### 理解置信度阈值

置信度阈值（Score Threshold）控制知识库检索结果的筛选标准：

- **低阈值（0.1-0.3）**：返回更多结果，但可能包含不相关内容
- **中等阈值（0.4-0.6）**：平衡相关性和数量，推荐使用
- **高阈值（0.7-0.9）**：只返回高度相关的结果，但可能遗漏相关信息

### 设置建议

- **通用场景**：推荐0.5，平衡准确性和覆盖率
- **高精度需求**：推荐0.7-0.8，确保结果高度相关
- **探索性搜索**：推荐0.3-0.4，获取更多相关信息

阈值设置会影响所有使用知识库的AI功能，包括AI对话、AI补全等。

## 知识库文件管理

### 添加文件到知识库

1. 在知识库管理页面，点击"添加文件"按钮
2. 选择要添加的文件（支持多种格式）
3. 系统会自动处理文件：
   - 转换文件为文本
   - 将文本分割成块
   - 生成向量嵌入
   - 存储到知识库

**支持的文件格式**：
- Markdown（.md）
- LaTeX（.tex）
- PDF（.pdf）
- Word（.docx）
- 图片（.png, .jpg等，通过OCR识别）
- 纯文本（.txt）

### 文件列表管理

知识库管理页面显示所有已添加的文件：

- **文件名**：显示文件的名称
- **状态**：显示文件是否启用
- **块数量**：显示文件被分割成的块数
- **向量数量**：显示生成的向量数量
- **操作**：提供文件管理操作

### 启用/禁用文件

您可以临时禁用某个文件，而不删除它：

1. 在文件列表中找到要操作的文件
2. 点击"启用"或"禁用"按钮
3. 禁用后，该文件不会被检索，但数据仍保留

**使用场景**：
- 临时排除某些文件
- 测试不同文件组合的效果
- 保留文件但暂时不使用

### 删除文件

1. 在文件列表中找到要删除的文件
2. 点击"删除"按钮
3. 确认删除操作

删除文件会：
- 删除文件记录
- 删除所有相关的数据块
- 删除所有相关的向量
- 操作不可恢复

**注意事项**：
- 删除操作不可恢复，请谨慎操作
- 删除大文件可能需要一些时间
- 删除后需要重新添加才能恢复

### 重命名文件

1. 在文件列表中找到要重命名的文件
2. 点击"重命名"按钮
3. 输入新的文件名
4. 确认重命名

重命名只改变显示名称，不影响文件内容和向量数据。

### 预览文件

您可以预览知识库中的文件内容：

1. 在文件列表中找到要预览的文件
2. 点击"预览"按钮
3. 查看文件的文本内容

预览功能可以帮助您：
- 确认文件内容是否正确
- 检查文件是否被正确处理
- 了解文件的文本结构

### 下载文件

您可以下载知识库中的文件：

1. 在文件列表中找到要下载的文件
2. 点击"下载"按钮
3. 选择保存位置

下载的文件是原始文件的副本，可以用于备份或分享。

## 向量重建

### 重建向量

如果文件的向量数据出现问题，或者更新了Embedding模型，可以重建向量：

1. 在文件列表中找到要重建的文件
2. 点击"重建向量"按钮
3. 等待重建完成

重建向量会：
- 重新处理文件文本
- 重新生成向量嵌入
- 更新向量索引

**使用场景**：
- 更换了Embedding模型
- 向量数据损坏
- 需要更新向量表示

### 重建全部向量

如果需要重建所有文件的向量：

1. 点击"重建全部向量"按钮
2. 确认操作
3. 等待所有文件重建完成

重建全部向量可能需要较长时间，特别是文件较多时。

## 知识库搜索测试

### 测试搜索功能

您可以在知识库管理页面测试搜索功能：

1. 在搜索框中输入查询文本
2. 点击"搜索"按钮
3. 查看搜索结果

搜索结果会显示：
- 匹配的文本片段
- 相似度分数
- 来源文件
- 上下文信息

### 搜索参数

测试搜索时可以调整：
- **查询文本**：输入要搜索的内容
- **返回数量**：设置返回结果的数量
- **阈值**：设置最低相似度阈值

## 清空知识库

### 清空所有数据

如果需要清空整个知识库：

1. 点击"清空知识库"按钮
2. 确认操作
3. 等待清空完成

清空知识库会：
- 删除所有文件记录
- 删除所有数据块
- 删除所有向量
- 操作不可恢复

**注意事项**：
- 清空操作不可恢复，请谨慎操作
- 清空前建议先备份重要文件
- 清空后需要重新添加文件

## 最佳实践

1. **文件组织**：按主题或项目组织文件，便于管理
2. **定期更新**：文件内容更新后，及时重建向量
3. **阈值调整**：根据实际使用效果调整置信度阈值
4. **文件清理**：定期删除不再需要的文件，保持知识库整洁
5. **备份重要文件**：重要文件建议先备份再添加到知识库

## 注意事项

1. **文件大小**：大文件处理时间较长，请耐心等待
2. **存储空间**：知识库会占用一定的存储空间
3. **处理时间**：添加文件和处理向量需要时间，请勿中断
4. **文件格式**：确保文件格式正确，否则可能无法处理
5. **网络连接**：使用API模式生成向量需要网络连接
