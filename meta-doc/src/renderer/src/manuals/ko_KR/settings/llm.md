# LLM 구성

## 개요

LLM(거대 언어 모델) 설정은 MetaDoc AI 기능의 핵심입니다. **Steam을 통해 배포되는 정식 빌드**는 예전 테스트 채널과 기본 권장 경로가 다릅니다.

- **Steam(권장)**: MetaDoc이 운영하는 **Cloudflare 기반 LLM API 프록시**(앱에서는 **「MetaDoc Cloud (Steam)」** 등으로 표시)를 사용합니다. **Steam으로 충전**하여 **크레딧(credits)** 을 구매한 뒤 AI 기능을 씁니다. 일반적으로 **서드파티 API 키를 직접 발급할 필요가 없습니다**.
- **자체 API(BYOK)**: **개발자/실험적** 옵션입니다. **LLM** 설정에서 **실험적 옵션**을 펼치고 **「실험적 연결 사용」** 을 켜야 기존과 같은 **다중 프로필 + 사용자 정의 API** 화면이 나타납니다. 자세한 내용은 아래 「실험적 연결과 BYOK」 및 링크된 문서를 참고하세요.

아래에서는 먼저 **Steam / MetaDoc Cloud**(잔액, 충전, 모델 전환)를 설명하고, 이어서 **실험적 옵션**, 마지막으로 **전체 UI 동작**(LLM 사용, 온도, 구성 그리드 등)을 설명합니다.

## Steam: MetaDoc Cloud 및 크레딧(권장)

**Steam**으로 설치·실행하는 MetaDoc용입니다.

### 요약 절차

1. **LLM 설정 열기**: **설정** → **LLM**(또는 메뉴에서 **설정**).
2. **「MetaDoc Cloud (Steam)」** 찾기: **잔액(credits)**, **모델** 선택, **충전**, **새로 고침** 등을 사용할 수 있습니다.
3. **충전**: **충전**을 눌러 **Steam 인앱 구매**로 credits를 구매합니다. Steam 라이브러리에서 실행하고 Steam 클라이언트가 필요합니다(구매가 안 되면 UI에 Steam/Greenworks 조건이 안내됩니다).
4. **잔액 확인**: 이 페이지에 credits가 표시됩니다. 트레이의 **Steam**에서 **계정 credits 잔액**을 보고 충전을 시작할 수 있습니다.
5. **요금**: **모델**마다 credits 차감 규칙이 다를 수 있습니다. 앱에 표시된 내용을 따르세요.
6. **모델 전환**: MetaDoc Cloud에서 **모델**을 고르면 **AI 채팅, 자동 완성, 교정, Agent** 등이 해당 모델을 사용합니다. 사용 중 언제든 바꿀 수 있습니다.

### 참고

- 엔드포인트와 프록시는 MetaDoc이 운영하므로, 이 경로에서는 OpenAI/DeepSeek 등 **URL·키를 직접 넣지 않아도** 됩니다(BYOK와 다름).
- 잔액 부족이나 네트워크 오류 시 AI 기능이 막히거나 오류가 날 수 있습니다. **충전** 또는 **잔액 새로 고침**을 시도하세요.

## 개발자 옵션: 실험적 연결과 BYOK(자체 API)

**Steam** 빌드에서 예전처럼 **직접 LLM API를 구성**하려면 **개발자/실험적** 경로를 씁니다. **LLM** 설정의 **실험적 옵션**을 연 뒤 **「실험적 연결 사용」** 을 켜면(확인 대화상자 있음) **다중 구성 카드, API 주소, API 키, 공급자 유형** 등이 표시됩니다.

**중요**:

- **실험적**이며 표준 Steam/MetaDoc Cloud 경험과 **다를 수 있고**, **서드파티에 직접 비용**이 나가거나 네트워크·규정 리스크가 커질 수 있습니다.
- **자체 API 사용 시 키 관리, 요금, 가용성, 약관 준수는 전적으로 사용자 책임**입니다. MetaDoc은 클라이언트 구성·연결만 제공합니다.

- **전역 설정**: LLM 사용 토글, 온도 슬라이더, 추론 태그 제거, 터미널 실행 기본 허용 등
- **구성 그리드**: 카드 형태로 모든 구성 표시. 카드를 클릭해 전환, 현재 구성은 녹색 테두리

데모 모드에서는 레이아웃만 확인할 수 있으며 저장되지 않습니다.

LLM을 켠 뒤 다음 기능을 사용할 수 있습니다:

- AI 채팅
- AI 교정
- AI 자동 완성
- AI 도우미
- Agent 프레임워크

**주의**:

- LLM 사용 시 API 호출로 비용이 발생할 수 있습니다
- 먼저 서비스를 구성한 뒤 켜는 것을 권장합니다
- AI가 필요 없으면 끄면 리소스를 절약할 수 있습니다

## LLM 온도

<SettingLlmSection mode="demo" />

### 온도 매개변수

온도는 생성 텍스트의 무작위성을 조절합니다:

- **낮음(0-0.5)**: 더 결정적이고 일관됨
- **중간(0.5-1.0)**: 대부분의 경우 균형
- **높음(1.0-2.0)**: 더 다양하고 창의적

### 권장

- **기술 문서**: 0.3-0.5
- **창작**: 0.7-1.0
- **코드**: 0.2-0.4
- **대화**: 0.7-0.9

온도는 AI 채팅, 자동 완성, 교정 등 모든 LLM 기능에 적용됩니다.

## 추론 태그 자동 제거

일부 LLM은 추론 과정을 특수 태그로 붙일 수 있습니다. 이 옵션을 켜면 MetaDoc이 태그를 걸러 최종 출력만 남깁니다. LLM이 태그를 지원하지 않으면 효과가 없습니다.

## 주의사항

1. **API 키 보안**: BYOK/실험적 연결 시 타인과 공유하지 마세요
2. **비용**: MetaDoc Cloud는 **credits**, BYOK는 공급자 요금
3. **네트워크**: 외부 API는 안정적인 연결이 필요합니다
4. **백업**: 중요한 구성은 내보내기로 백업하세요
5. **모델**: 모델마다 능력과 제한이 다릅니다
