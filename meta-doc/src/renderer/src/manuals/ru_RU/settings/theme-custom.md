# Управление пользовательскими темами

## Обзор

Управление пользовательскими темами позволяет создавать, редактировать, удалять и копировать пользовательские темы. С помощью пользовательских тем вы можете настроить внешний вид интерфейса в соответствии с личными предпочтениями, улучшив опыт использования.

## Создание новой пользовательской темы

### Создание новой темы

1.  На странице настроек тем нажмите на карточку "Новая тема" (значок +)
2.  Во всплывающем диалоговом окне:
    -   Введите название темы (необязательно, по умолчанию используется значение цвета)
    -   Выберите цвет темы (используя палитру цветов)
3.  Нажмите кнопку "Сохранить"

Вы можете получить доступ к настройкам тем через верхнюю строку меню:

### Выбор цвета темы

Палитра цветов предоставляет следующие возможности:

-   **Выбор цвета**: нажмите на область цвета для выбора
-   **Предустановленные цвета**: выбор из списка предустановленных цветов
-   **Регулировка прозрачности**: настройка прозрачности цвета (альфа-канал)
-   **Ввод значения цвета**: прямое введение HEX-значения цвета

### Наименование темы

-   **Автоматическое наименование**: если название не введено, система использует значение цвета в качестве названия
-   **Пользовательское название**: введите осмысленное название для удобства идентификации и управления
-   **Рекомендации по наименованию**: используйте описательные названия, такие как "Рабочая тема", "Ночной режим" и т.д.

## Редактирование пользовательской темы

### Изменение темы

1.  В списке тем найдите пользовательскую тему, которую хотите отредактировать
2.  Нажмите кнопку "Ещё" (три точки) на карточке темы
3.  Выберите "Редактировать"
4.  В диалоговом окне измените название или цвет темы
5.  Нажмите кнопку "Сохранить"

<DialogDemo mode="demo" dialogType="theme-edit" />

### Быстрое редактирование цвета

Вы также можете редактировать цвет прямо на карточке темы:

1.  Нажмите на палитру цветов на карточке темы
2.  Выберите новый цвет
3.  Цвет будет применён немедленно

**Важные моменты**:

-   Предустановленные темы нельзя редактировать
-   Редактировать можно только пользовательские темы
-   После редактирования необходимо сохранить изменения, чтобы они вступили в силу навсегда

## Удаление пользовательской темы

### Удаление темы

1.  В списке тем найдите пользовательскую тему, которую хотите удалить
2.  Нажмите кнопку "Ещё" (три точки) на карточке темы
3.  Выберите "Удалить"
4.  Подтвердите действие удаления

**Важные моменты**:

-   Удаление необратимо
-   Если удаляется текущая активная тема, система автоматически переключится на тему по умолчанию
-   Предустановленные темы нельзя удалить

## Копирование темы

### Копирование существующей темы

1.  В списке тем найдите тему, которую хотите скопировать
2.  Нажмите кнопку "Ещё" (три точки) на карточке темы
3.  Выберите "Копировать"
4.  Система создаст копию, добавив к названию " (копия)"
5.  Копию можно отредактировать для создания новой темы

### Сценарии использования

-   **Создание новой темы на основе существующей**: скопируйте и измените цвет
-   **Создание варианта темы**: создайте похожие, но слегка отличающиеся темы
-   **Резервное копирование темы**: скопируйте тему в качестве резервной копии

## Настройка цвета темы

### Функции палитры цветов

Палитра цветов предоставляет широкие возможности выбора цвета:

-   **Цветовая панель**: нажмите для выбора цвета
-   **Предустановленные цвета**: быстрый выбор часто используемых цветов
-   **Ввод значения цвета**: прямое введение значений в форматах HEX, RGB, HSL и др.
-   **Регулировка прозрачности**: настройка прозрачности цвета

<DialogDemo mode="demo" dialogType="color-picker" />

### Предустановленные цвета

MetaDoc предоставляет множество предустановленных цветов:

-   **Основные цвета**: красный, оранжевый, жёлтый, зелёный, голубой, синий, фиолетовый, серый
-   **Светлые оттенки**: светло-красный, светло-оранжевый, светло-жёлтый и т.д.
-   **Тёмные оттенки**: тёмно-красный, тёмно-оранжевый, тёмно-жёлтый и т.д.

### Форматы цвета

Поддерживаемые форматы цвета:

-   **HEX**: `#FF5733` (наиболее распространённый)
-   **RGB**: `rgb(255, 87, 51)`
-   **HSL**: `hsl(9, 100%, 60%)`

## Применение темы

### Применение пользовательской темы

1.  В списке тем нажмите на карточку пользовательской темы, которую хотите использовать
2.  Тема будет применена немедленно
3.  Цвета интерфейса будут автоматически сгенерированы на основе цвета темы

### Влияние цвета темы

Цвет темы влияет на следующие элементы интерфейса:

-   **Фоновый цвет**: основной и второстепенный фон
-   **Цвет текста**: основной и второстепенный текст
-   **Боковая панель**: фон и текст боковой панели
-   **Редактор**: фон редактора и панель инструментов
-   **Другие элементы**: кнопки, границы, подсветка и т.д.

### Автоматическая цветовая схема

MetaDoc автоматически генерирует цветовую схему на основе цвета темы:

-   **Светлая тема**: при светлом цвете темы генерируется светлая цветовая схема
-   **Тёмная тема**: при тёмном цвете темы генерируется тёмная цветовая схема
-   **Алгоритм цветовой схемы**: используется смешивание цветов и регулировка насыщенности

## Управление темами

### Список тем

На странице настроек тем отображаются все доступные темы:

-   **Предустановленные темы**: встроенные в систему темы
-   **Пользовательские темы**: темы, созданные пользователем
-   **Текущая тема**: отображается маркер выбора

### Сортировка тем

Темы отображаются в следующем порядке:

1.  Синхронизированная системная тема (следует за системой)
2.  Предустановленные светлые/тёмные темы
3.  Пользовательские темы (по времени создания)

### Состояние темы

На каждой карточке темы отображается:

-   **Предпросмотр цвета темы**: показывает основной цвет темы
-   **Название темы**: показывает название темы
-   **Значение цвета**: показывает HEX-значение цвета
-   **Маркер выбора**: текущая используемая тема

## Рекомендации

1.  **Наименование тем**: используйте осмысленные названия для удобства идентификации
2.  **Выбор цвета**: выбирайте цвета, щадящие зрение, избегайте слишком ярких
3.  **Резервное копирование тем**: рекомендуется копировать важные темы в качестве резервных копий
4.  **Регулярная очистка**: удаляйте неиспользуемые темы для поддержания порядка в списке
5.  **Тестирование эффекта**: после создания темы протестируйте фактический эффект и настройте на основе опыта использования

## Важные моменты

1.  **Предустановленные темы**: предустановленные темы нельзя редактировать или удалять
2.  **Совместимость тем**: некоторые темы могут отображаться по-разному в различных средах
3.  **Выбор цвета**: рекомендуется выбирать цвета с умеренной контрастностью для обеспечения читаемости
4.  **Количество тем**: рекомендуется не создавать слишком много тем для поддержания простоты списка
5.  **Синхронизация тем**: изменения тем синхронизируются между всеми окнами
