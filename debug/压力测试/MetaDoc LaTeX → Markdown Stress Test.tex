# 二、LaTeX 压力测试样例

```tex
\documentclass{article}

\usepackage{amsmath}
\usepackage{hyperref}
\usepackage{graphicx}

\title{MetaDoc LaTeX → Markdown Stress Test}
\author{Test System}
\date{\today}

\begin{document}

\maketitle

\section{Basic Formatting}

Normal text with \textbf{bold}, \textit{italic}, and nested formatting
\textbf{bold and \textit{italic inside}}.

Edge case:

\textit{italic with \textbf{bold inside}}

Inline code style:

\texttt{const x = 10;}

\section{Links and References}

Simple link:

\href{https://openai.com}{OpenAI}

Formatted link:

\href{https://example.com}{\textbf{Bold Link}}

\section{Lists}

Unordered list:

\begin{itemize}
\item First item
\item Second item
\item Third item
\end{itemize}

Nested list:

\begin{itemize}
\item Level 1
\begin{itemize}
\item Level 2
\begin{itemize}
\item Level 3
\end{itemize}
\end{itemize}
\end{itemize}

Ordered list:

\begin{enumerate}
\item First
\item Second
\item Third
\end{enumerate}

Mixed formatting inside list:

\begin{itemize}
\item \textbf{Bold item}
\item \textit{Italic item}
\item \texttt{Code item}
\end{itemize}

\section{Code Blocks}

Verbatim example:

\begin{verbatim}
function hello() {
    console.log("Hello world");
}
\end{verbatim}

Code containing Markdown-like syntax:

\begin{verbatim}
# Heading
**bold**
*italic*
\end{verbatim}

\section{Mathematics}

Inline math:

$E = mc^2$

More complex inline math:

$\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$

Displayed equation:

\[
E = mc^2
\]

More complex display math:

\[
\frac{d}{dx} \left( \int_0^x f(t)\,dt \right) = f(x)
\]

\section{Images}

Image example:

\includegraphics{example-image.png}

\section{Nested Formatting Stress Test}

\textbf{bold \textit{italic \texttt{code inside}}}

Another complex example:

\textit{italic \textbf{bold \texttt{code}}}

\section{Special Characters}

Special characters:

\# \$ \% \& \_ \{ \}

Backslash:

\texttt{\textbackslash}

\section{Long Paragraph}

Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
\textbf{Vestibulum \textit{interdum}} nisl sed erat fermentum,
sed posuere \texttt{nulla} facilisis.

\section{Mixed Content}

\begin{itemize}
\item \textbf{Bold item with math $E=mc^2$}
\item \textit{Italic item with \texttt{code}}
\item Link item: \href{https://example.com}{Example}
\end{itemize}

Quote-like text:

\begin{quote}
Quote with \textbf{bold}, \textit{italic}, and math $a^2+b^2=c^2$.
\end{quote}

\end{document}
